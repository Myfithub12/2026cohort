Baseball Scorecard

My general idea is to create a baseball scorecard for fans and teams to keep track. I am an avid baseball fan and have always kept score by hand. With this app it will make keeping score easier with eventual calculations and daily updates to player stats. By starting this program with a few simple concepts like entering player names and jersey numbers, field position, and selecting how the player at bat is reacts when the pitcher throws the ball like walks, outs, and the different ways of getting on base like hitting the ball, being hit by the ball, errors, and other ways. I have created a few drop down boxes, an add player button to add new players, and an export to notepad button to save a personal copy of a scorecard to ones computer to look at later. A fun thing about exporting to notepad is that it saves the file as a date for future reference.

I have selected three requirements for this project:
-Persist important data to the user to local storage and make the stored data accessible in your app. (including after reload/refresh)
-Calculate and display data based on an external factor (e.g.: Get the current date)
-Create a function that accepts two or more input parameters and returns a value that is calculated or determined by the inputs. (Basic math functions don't count)
